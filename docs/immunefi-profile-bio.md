## Registration checklist

1. Open https://bugs.immunefi.com/ or https://immunefi.com/ → Sign up / Log in
2. Complete researcher profile (name, bio above, GitHub)
3. Enable 2FA on the email/GitHub you use
4. Prefer **NO-KYC** programs first while portfolio is young (filter on program pages)
5. Read each program’s scope, out-of-scope, and PoC rules before testing
6. Keep notes: program URL, in-scope assets, block/commit hash, local fork RPC
